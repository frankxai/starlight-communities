# Security And Privacy

## Never Commit

- Real member profiles.
- Contact lists.
- Raw voice or video.
- Private reflections.
- Matching rationale involving sensitive personal data.
- API keys, tokens, cookies, `.env` files, or provider secrets.

## Human Gates

Human approval is required for:

- external invites;
- direct messages;
- public artifact publishing;
- calendar events with external attendees;
- raw voice/video storage;
- leaderboard publication;
- paid services or subscriptions.

## Blocked Use

This project does not provide legal, financial, medical, or mental-health
advice. Quests can help members map ideas, commitments, and artifacts, but
professional advice must stay outside the agent system.

## Third-Party Skills

Third-party agent skills and templates are research inputs, not trusted runtime
dependencies. Review permissions, provenance, license, and behavior before
adapting anything into a production cell.
