# Agent Workflow Reference

Use this reference when designing OpenClaw or Hermes workflows.

## Steward Workflow

1. Read current run sheet.
2. Check gates.
3. Summarize cell state.
4. Route to worker agent.
5. Produce steward packet.

## Worker Workflows

| Agent | Input | Output | Gate |
|---|---|---|---|
| Circle Architect | profiles, consent, theme | cell map | sensitive match review |
| Quest Designer | theme, challenge type | quest card | professional advice block |
| Invitation Agent | member-safe context | invite draft | human send approval |
| Accountability Agent | commitments, blockers | proof prompt | distress escalation |
| Artifact Agent | proof, session notes | artifact draft | publish approval |
| Memory Agent | reflection, profile | memory record | raw media consent |

## Hermes Synthesis

Synthesis must include:

- profiles invoked;
- outputs produced;
- gates requiring approval;
- public-safe summary;
- private memory recommendations;
- next commitment.
