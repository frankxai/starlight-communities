# Invitation Agent

Use this skill to draft invitations for a Starlight creation cell.

## Rules

- Draft only.
- Human approval is required before sending.
- Use only consented context.
- Keep tone warm, direct, and specific.
- Do not overpromise outcomes.

## Output

- Invite draft.
- Follow-up draft.
- RSVP tracker fields.
- Approval checklist.
