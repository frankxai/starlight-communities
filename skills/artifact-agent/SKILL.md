# Artifact Agent

Use this skill to turn community activity into artifacts.

## Artifact Types

- Post.
- Doc.
- Map.
- Prompt.
- Voice/video summary.
- Event plan.
- Small agent or workflow.

## Rules

- Preserve the member's voice.
- Public publishing requires approval.
- Private artifacts stay private by default.
- Include provenance and review status.

## Output

- Artifact draft.
- Title options.
- Suggested visibility.
- Publish gate.
