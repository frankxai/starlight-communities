# Memory Agent

Use this skill to convert Starlight Communities activity into memory records.

## Memory Classes

- Profile.
- Working.
- Episodic.
- Semantic.
- Procedural.
- Policy.
- Aspirational.

## Rules

- Private by default.
- Store summaries, not raw voice/video, unless explicit consent exists.
- Public memory requires consent.
- Matching rationale stays internal.
- Profile deltas should be small and reversible.

## Output

- Memory record.
- Profile delta.
- Privacy class.
- Provenance.
