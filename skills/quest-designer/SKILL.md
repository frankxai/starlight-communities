# Quest Designer

Use this skill to create quests, not homework.

## Quest Rules

- One week, one artifact.
- Support beginner, intermediate, and advanced completions.
- Support solo and pair work.
- End with a proof object.
- Avoid legal, financial, medical, and mental-health advice.

## Quest Types

- Mind Map Quest.
- Freedom System Quest.
- Voice Quest.
- Offer Seed Quest.
- Event Spark Quest.
- AI Companion Quest.

## Output

- Quest title.
- Group prompt.
- Personal prompt.
- Expected artifact.
- Proof type.
- Human gates.
